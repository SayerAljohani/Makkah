# Makkah

- How to build project 
- download donet core 3 that run these commands
- لتشغيل المشروع حمل النسخه الثالثه من donet core 
- وبعد ذلك استخدم التعليمات التالية لتشغيل المشروع 
 1. dotnet restore 
 2. dotnet build 
 3. dotnet watch run
